# ПРИЛОЖЕНИЕ НАХОДИТСЯ В ALPHA ВЕРСИИ, ИСПОЛЬЗУЙТЕ НА СВОЙ СТРАХ И РИСК.

**PWA-тренажёр для изучения японского языка по учебнику Genki**

Kitsune Genki — это прогрессивное веб-приложение (PWA) для изучения японского языка, которое можно установить на любое устройство и использовать оффлайн. Приложение построено на системе интервальных повторений и геймификации для эффективного запоминания материала.

---

## ✨ Основные возможности

- 📚 **Система интервальных повторений (SRS)** — умный алгоритм для долгосрочного запоминания
- 🔥 **Ежедневные стрики** — мотивация для регулярных занятий
- ⭐ **XP и система уровней** — отслеживание прогресса и роста
- 🪙 **Игровая валюта** — зарабатывайте монеты за выполнение заданий
- 📅 **План обучения** — составьте персональное расписание изучения глав
- ⚡ **Квесты и челленджи** — выполняйте задания и получайте награды
- 🏆 **Достижения** — коллекционируйте достижения за различные успехи
- 📖 **Интерактивные истории** — изучайте кандзи через увлекательные истории
- ✍️ **Тренировка письма** — практикуйте написание иероглифов с помощью HanziWriter
- 🌙 **Тёмная тема** — удобный режим для занятий в любое время суток
- 📱 **Оффлайн-режим** — работает без интернета после установки

---

## 🚀 Установка приложения

Kitsune Genki можно установить как нативное приложение на любое устройство. После установки оно будет доступно с главного экрана и работать оффлайн.

### 📱 Chrome (Android)

1. Откройте приложение в браузере Chrome
2. Нажмите на **три точки** (⋮) в правом верхнем углу
3. Выберите **"Добавить на главный экран"** или **"Установить приложение"**
4. Подтвердите установку

### 🍎 Safari (iOS)

1. Откройте приложение в браузере Safari
2. Нажмите на кнопку **"Поделиться"** (квадрат со стрелкой вверх) внизу экрана
3. Прокрутите вниз и выберите **"На экран «Домой»"**
4. Нажмите **"Добавить"** в правом верхнем углу
5. Приложение появится на главном экране

### 🦊 Firefox (Android)

1. Откройте приложение в браузере Firefox
2. Нажмите на **три точки** (⋮) в правом верхнем углу
3. Выберите **"Установить"**
4. Подтвердите установку

### 🔴 Opera (Android)

1. Откройте приложение в браузере Opera
2. Нажмите на **меню** (⋮)
3. Выберите **"Главная страница"** → **"Добавить на главный экран"**

## 🛠️ Технологии

Приложение создано с использованием современных веб-технологий:

- **HTML5** — структура приложения
- **CSS3** — стилизация и анимации
- **Vanilla JavaScript (ES6+)** — вся логика приложения без фреймворков
- **PWA (Progressive Web App)** — установка и оффлайн-работа
- **Service Worker** — кэширование для работы без интернета
- **HanziWriter** — библиотека для визуализации и практики написания иероглифов
- **Web Storage API** — локальное хранение данных пользователя

## 📋 Требования

- Современный веб-браузер с поддержкой PWA:
  - Chrome 67+
  - Firefox 79+
  - Safari 11.1+
  - Edge 79+
  - Opera 54+
- JavaScript должен быть включён
- Для установки как PWA требуется HTTPS (или localhost для разработки)

## 🎯 Как использовать

1. **Установите приложение** по инструкции выше
2. **Выберите главы** для изучения в разделе "План обучения"
3. **Изучайте карточки** — запоминайте новые слова и иероглифы
4. **Повторяйте материал** — система SRS подскажет, когда пора повторить
5. **Выполняйте квесты** — получайте дополнительный опыт и награды
6. **Собирайте достижения** — открывайте новые значки за успехи
7. **Поддерживайте стрик** — занимайтесь каждый день для максимального эффекта

## 🎮 Система прогрессии

### Опыт (XP)
- Получайте XP за изучение и повторение карточек
- Повышайте уровень для открытия новых возможностей
- Отслеживайте свой прогресс в профиле

### Монеты 🪙
- Зарабатывайте монеты за выполнение заданий
- Используйте для покупок в магазине (будущая функция)

### Стрики 🔥
- Занимайтесь каждый день для поддержания стрика
- Решайте минимум 10 карточек в день
- Длинные стрики дают бонусные награды

### Ранги
- Начните с ранга Alpha
- Прогрессируйте через Beta, Gamma к Delta
- Каждый ранг имеет 12 уровней
