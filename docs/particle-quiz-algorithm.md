# Алгоритм генерации предложений для Particle Quiz

## Обзор

Система генерации предложений для изучения японских частиц (N5) использует **двухуровневую архитектуру**:

1. **Готовые предложения (Curated Sentences)** — приоритетный уровень
2. **Умные шаблоны (Smart Templates)** — резервный уровень с интеллектуальной генерацией

---

## Проблема, которую решает система

### Исходная проблема

Старый алгоритм генерировал предложения через простую подстановку слов в шаблоны:

```javascript
// ❌ ПРОБЛЕМА: Дублирование переводов
template: (w1, w2) => `${w1.writing} [ _ ] ${w2.writing} です`;
hint: (w1, w2) => `${shortT(w1)} — это ${shortT(w2)}`;
```

**Результат для これ (это) + 本 (книга):**

- Предложение: `これ [ _ ] 本 です`
- Подсказка: `"это — это книга"` ❌ (дублирование слова "это")

### Корневая причина

Местоимения (これ, 私, あなた) имеют переводы ("это", "я", "ты"), которые дублируются при наивной склейке в русской подсказке.

---

## Архитектура решения

```
generateParticleQuiz(particle)
    ↓
    ├─→ ШАГ 1: CURATED_PARTICLE_SENTENCES[particle]
    │   └─→ Возвращает готовое естественное предложение
    │
    └─→ ШАГ 2 (fallback): SMART_PARTICLE_TEMPLATES[particle]
        ├─→ Подбор слов из словаря с фильтрацией
        ├─→ smartHint() — интеллектуальная генерация подсказки
        └─→ Возвращает сгенерированное предложение
```

## Уровень 2: Умные шаблоны

### Smart Hint — Интеллектуальная генерация подсказок

```javascript
function smartHint(word1, word2, grammarQuestion, particleType) {
  const t1 = shortT(word1); // "это", "я", "друг"
  const t2 = shortT(word2); // "книга", "студент"

  // ПРАВИЛО 1: Частицы は/も с местоимениями
  if ((particleType === 'は' || particleType === 'も') && isPronoun(word1)) {
    if (particleType === 'は') {
      return `${t1} — ${t2}`; // "это — книга" ✅ (без дубля "это — это книга")
    } else if (particleType === 'も') {
      return `${t1} — тоже ${t2}`; // "я — тоже студент" ✅
    }
  }

  // ПРАВИЛО 2: Частица の с местоимениями (родительный падеж)
  if (particleType === 'の' && isPronoun(word1)) {
    const genitiveMap = {
      я: 'моя/мой/моё',
      ты: 'твоя/твой/твоё',
      это: 'этого',
    };
    const genitive = genitiveMap[t1] || t1;
    return `${t2} (чья/чей/чьё? — ${genitive})`; // "книга (чья? — моя)" ✅
  }

  // ПРАВИЛО 3: Места-местоимения (ここ, そこ)
  if (isLocationPronoun(word1) && grammarQuestion) {
    return `${t2} (${grammarQuestion})`; // "учиться (где?)" без дубля "здесь"
  }

  // ПРАВИЛО 4: Стандартный формат
  if (grammarQuestion) {
    return `${t2} (${grammarQuestion} — ${t1})`; // "читать (что? — книгу)"
  }

  return `${t1} — ${t2}`;
}
```

### Фильтрация запрещенных комбинаций

```javascript
function isProhibitedCombination(word1, word2, particleType) {
  // Запрет 1: местоимение + местоимение (кроме の)
  if (particleType !== 'の' && isPronoun(word1) && isPronoun(word2)) {
    return true; // ❌ "私 は これ です" — бессмысленно
  }

  // Запрет 2: место-местоимение + место-местоимение
  if (isLocationPronoun(word1) && isLocationPronoun(word2)) {
    return true; // ❌ "ここ で そこ" — нелогично
  }

  // Запрет 3: одинаковые слова
  if (word1.id === word2.id) {
    return true; // ❌ "本 を 本" — абсурд
  }

  return false;
}
```

## Покрытие частиц N5

### Базовые частицы (Genki 1-6)

| Частица | Функция           | Пример готового предложения                          |
| ------- | ----------------- | ---------------------------------------------------- |
| **は**  | Тема              | `田中さん [ _ ] 学生 です` → "Танака-сан — студент"  |
| **の**  | Принадлежность    | `先生 [ _ ] 本` → "книга (чья? — учителя)"           |
| **を**  | Прямой объект     | `コーヒー [ _ ] 飲みます` → "пить (что? — кофе)"     |
| **に**  | Направление/время | `学校 [ _ ] 行きます` → "идти (куда? — в школу)"     |
| **で**  | Место действия    | `学校 [ _ ] 勉強します` → "учиться (где? — в школе)" |
| **が**  | Субъект           | `猫 [ _ ] います` → "кошка — есть/находится"         |

### Дополнительные частицы (Genki 7-12)

| Частица  | Функция                  | Пример                                                      |
| -------- | ------------------------ | ----------------------------------------------------------- |
| **と**   | Совместность             | `友達 [ _ ] 話します` → "говорить (с кем? — с другом)"      |
| **へ**   | Направление (формальное) | `学校 [ _ ] 行きます` → "идти (куда? — в школу)"            |
| **から** | Исходная точка           | `学校 [ _ ] 帰ります` → "возвращаться (откуда? — из школы)" |
| **まで** | Конечная точка           | `学校 [ _ ] 歩きます` → "идти пешком (докуда? — до школы)"  |
| **も**   | Тоже                     | `田中さん [ _ ] 学生 です` → "Танака-сан — тоже студент"    |
| **か**   | Вопрос                   | `学生 です [ _ ]` → "студент — это студент?"                |

## Логика выбора предложения

### Алгоритм в `generateParticleQuiz()`

```javascript
function generateParticleQuiz(particle, lessonData, state, LESSONS) {
  // ШАГ 1: Приоритет — готовые предложения
  const curatedSentences = CURATED_PARTICLE_SENTENCES[particle];
  if (curatedSentences && curatedSentences.length > 0) {
    const example = curatedSentences[Math.floor(Math.random() * curatedSentences.length)];
    return {
      sentence: example.sentence,
      correctParticle: example.correct,
      options: generateDistractors(particle, state, LESSONS),
      russianHint: example.hint,
      words: [],
    };
  }

  // ШАГ 2: Fallback — умные шаблоны
  const templateDef = SMART_PARTICLE_TEMPLATES[particle];
  const selectedWords = selectWordsForSlots(templateDef.slots, state, LESSONS);

  if (!selectedWords) {
    // ШАГ 3: Последний fallback — откат к готовым предложениям
    return useCuratedFallback(particle, state, LESSONS);
  }

  return {
    sentence: templateDef.template(...selectedWords),
    correctParticle: particle,
    options: generateDistractors(particle, state, LESSONS),
    russianHint: templateDef.hint(...selectedWords),
    words: selectedWords,
  };
}
```

### Пример 1: Частица は с местоимением

**Входные данные:**

- `particle = 'は'`
- Слова: `これ` (это), `本` (книга)

**Старая система:**

```javascript

// Результат: "это — это книга" ❌
```

**Новая система (Smart Hint):**

```javascript
if (particleType === 'は' && isPronoun(word1)) {
  return `${t1} — ${t2}`; // "это — книга" ✅
}
```

### Пример 2: Частица の с местоимением

**Входные данные:**

- `particle = 'の'`
- Слова: `私` (я), `本` (книга)

**Старая система:**

```javascript
hint: (w1, w2) => `${shortT(w2)} (чей? — ${shortT(w1)})`;
// Результат: "книга (чей? — я)" ❌ (грамматически неверно)
```

**Новая система (Smart Hint):**

```javascript
if (particleType === 'の' && isPronoun(word1)) {

  return `${t2} (чья/чей/чьё? — ${genitive})`;

}
```

## Преимущества системы

### 1. Надежность

- **Нет дублирования переводов** — Smart Hint учитывает особенности местоимений
- **Нет абсурдных комбинаций** — фильтр `isProhibitedCombination()`

### 2. Естественность

- **Приоритет готовых предложений** — экспертная проверка
- **Контекстные подсказки** — грамматически правильные вопросы в русском языке

### 3. Масштабируемость

- **Двухуровневая архитектура** — готовые примеры + шаблоны
- **Легко расширять** — добавить новую частицу = добавить 5-8 примеров

### 4. Обучающий эффект

- **Разнообразие примеров** — каждая частица имеет 5-9 вариантов
- **Прогрессия сложности** — от базовых слов (Genki 1-3) к сложным (Genki 10-12)
